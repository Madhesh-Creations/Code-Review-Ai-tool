# CodeLens — AI Code Review Tool

An intelligent, browser-based code reviewer powered by Claude AI. Paste your code and get instant, structured feedback on bugs, security vulnerabilities, performance bottlenecks, and style issues — with zero backend, zero cost.

## Features
- 🐛 Bug detection with line-level references
- 🔒 Security vulnerability scanning (SQL injection, hardcoded secrets, XSS, etc.)
- ⚡ Performance improvement suggestions
- ✨ Best practices & style feedback
- 📊 Quality score (0–100) with a stats breakdown
- 9+ languages supported with auto-detection
- Runs 100% in the browser — no server, no database

## Tech Stack
- Vanilla HTML / CSS / JavaScript
- Anthropic Claude API (`claude-sonnet`)
- Deployed on GitHub Pages (free)

## How to Use
1. Get a free API key at [console.anthropic.com](https://console.anthropic.com)
2. Paste your code in the editor
3. Enter your API key and click **Analyze Code**
